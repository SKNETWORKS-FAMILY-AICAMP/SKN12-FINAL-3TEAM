## 🔍 프로젝트 개요

<div align="center">
  <table>
    <tr>
      <td align="center">
        <h3>목표</h3>
        <p>AI 기반 회의 분석 및<br/>스마트 업무 분담 플랫폼</p>
      </td>
      <td align="center">
        <h3>해결 과제</h3>
        <p>수동 회의록 작성 시간 80% 절감<br/>자동 업무 배정 최적화</p>
      </td>
      <td align="center">
        <h3>혁신</h3>
        <p>LLM 기반 지능형 분석<br/>실시간 협업 도구 연동</p>
      </td>
    </tr>
  </table>
</div>

기존 회의록 생성 기능은 단순 기록에 그쳐 업무 효율화에 한계가 있습니다. 

**TtalKkak**은 대규모 언어모델(LLM)을 활용하여 회의 내용을 분석하고, 업무의 복잡도·팀원의 역량·가용 시간을 고려한 최적의 업무 분담을 자동으로 수행합니다. 

Slack, Notion, Jira 등 협업 도구와의 실시간 연동 및 자동화를 통해 중복 작업을 줄이고, 팀 전체의 협업 효율성을 극대화합니다.
